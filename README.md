# 🚀 AWESOME First Project: *Eniac's Expansion to Brazil*

## 📌 Overview

This repository contains the first project completed as part of a **Data Science course**, focusing on **MySQL** and **Tableau**.

The project explores strategic business questions related to *Eniac’s potential expansion into Brazil* through a partnership with Magist.

---

## 🎯 Project Objective

The main goal of this project is to analyze whether Magist is a suitable partner for Eniac by answering two key business questions:

1. **Product Compatibility**
   *Eniac’s catalogue is 100% tech products and heavily based on Apple-compatible accessories. Is Magist a suitable partner for such high-end tech products?*

2. **Delivery Performance**
   *Fast delivery is crucial for customer satisfaction. While Magist offers low delivery fees through the public Post Office, are deliveries fast enough to meet expectations?*
